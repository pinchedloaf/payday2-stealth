payday2-stealth
---------------
Payday 2 mod for easy stealthing.

Requires
--------
IPHLPAPI.dll, v.3.2.0 2015-07-15:1121 version (included).
- Original Author: sirgoodsmoke

Installation

Just copy to pd2 directory. Press F3 for easy stealth.
